# Frontend_Tutorial

網頁前端工程教學

## 編輯器

選用 VSCode (微軟開發)

編輯器擴充：

-   Open in Default Browser
-   Prettier - Code formatter
-   TabNine
-   ESLint

編輯器設定：

```
"editor.formatOnSave": true,
"[javascript]": {
    "editor.formatOnSave": true,
    "editor.defaultFormatter": "esbenp.prettier-vscode"
},
```
